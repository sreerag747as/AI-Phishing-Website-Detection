# AI-Powered Phishing Website Detection

## Overview
This project detects phishing websites using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Machine Learning Model
Random Forest Classifier

## Features
- Dataset preprocessing
- Model training
- Website prediction
- URL testing
- Visualization graphs

## Dataset
Kaggle Phishing Website Dataset

## Result
Successfully classified phishing and legitimate websites with high accuracy.
